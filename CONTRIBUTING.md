# Contributing

Thanks for helping improve the NLC Cyber Defense resource hub.

## Rules (keep it safe)
- **No sensitive data**: no credentials, internal IPs, student PII, access details, or private screenshots.
- **No CTF secrets**: writeups are welcome, but do not publish flags or anything that breaks event rules.
- **No copyrighted/restricted uploads**: do not add commercial training PDFs, certification guides, or paid course materials unless you have explicit redistribution permission.
- Keep content educational and reproducible.

## What to contribute
- Lab instructions improvements (clarity, troubleshooting)
- Tool setup notes
- Diagrams and summaries
- Sanitized practice writeups

## How to contribute
1) Fork the repo
2) Create a branch: `feature/<short-name>`
3) Make changes
4) Open a PR with:
   - What changed
   - Why it helps
   - Any references

## Style
- Prefer Markdown
- Use clear headings
- Keep steps short and verifiable
