How to contribute (safe + helpful)

  • Read: CONTRIBUTING.md
  • Allowed: clarifications, screenshots/diagrams (no sensitive data), tool notes, writeups without flags/secrets
  • Not allowed: credentials, private data, restricted school systems details

If you’re stuck, file an issue or submit a PR with a small improvement.
