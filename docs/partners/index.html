Community Partners / Employers

Ways to support NLC Cyber and connect with students.

High-impact ways to engage

  • Guest lecture: real-world case studies (IR, cloud security, SOC work, GRC)
  • Mentorship: resume review + mock interviews + portfolio feedback
  • Project ideas: small scoped problems students can build into a portfolio
  • Internships / jobs: structured, skill-building roles (even part-time)

Capstone projects (inquiries welcome)

We run supervised, portfolio-driven capstone work aligned to real community needs. If you can provide scope, mentorship, tooling, or funding, we welcome collaboration.

  • Regional SOC pilot: a hybrid model (student interns + paid professionals) to support K–12 ISDs with triage, detection tuning, and reporting.
  • Security assessments: structured assessments for local organizations and smaller municipalities, producing a prioritized findings report and a practical remediation roadmap.

Looking for: assessment opportunities, offers of employment/internships, mentorship, sponsorship, and other support.
