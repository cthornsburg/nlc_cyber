# NighHax (CTF Club) — Getting Started

NighHax is the student club supporting hacking-style / capture-the-flag (CTF) competitions.

## Quick start
- Pick 1–2 tracks to focus on initially:
  - Web
  - Crypto
  - Forensics
  - Reverse engineering
  - Linux / scripting
  - Blue team / detection

## Practice resources
See: `ctf/practice-resources.md`

## Writeups
If you publish writeups, keep them:
- educational,
- reproducible,
- and compliant with competition rules (no flags/secrets).
