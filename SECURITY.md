# Security Policy

## Reporting a security or privacy concern

If you believe you have found a security issue, privacy issue, or sensitive information that should not be public (for example: credentials, student PII, internal IPs, restricted system details, or unredacted screenshots), **please do not open a public GitHub Issue**.

Instead, email the maintainer:
- ithornsburg@alamo.edu

Include:
- A link to the file/page (or screenshot if needed)
- What you observed
- Steps to reproduce (if applicable)
- Any suggested fix
