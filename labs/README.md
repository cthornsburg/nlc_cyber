# Labs (Hands-On)

This folder contains hands-on labs used in the NLC Cyber Defense curriculum.

## How to use
- Start with the lab prerequisites (when present).
- Follow steps in order.
- Capture evidence/screenshots as instructed.
- Write a short reflection: what you observed, what broke, what you learned.

## Student expectations
- Work ethically and only in authorized lab environments.
- Do not attempt techniques on systems you do not own or have explicit permission to test.
- Do not upload sensitive data, credentials, or student PII to this repo.
