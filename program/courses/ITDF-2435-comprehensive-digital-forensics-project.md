# ITDF-2435 — Comprehensive Digital Forensics Project (Course Card)

## What this is
A capstone-style forensics project course focused on applying digital forensics end-to-end: scoping, acquisition/collection strategy, analysis, reporting, and presentation.

## What students produce
- A structured investigation plan
- Evidence inventory + chain-of-custody style notes (mock/sanitized)
- Findings report (executive summary + technical appendix)
- Brief-out presentation (10–15 minutes)

## Repo resources
- Forensics datasets: `labs/forensics/datasets.md`

## Notes
- Use sanitized datasets and mock scenarios only.
- Writeups should be reproducible and should not include sensitive content.
