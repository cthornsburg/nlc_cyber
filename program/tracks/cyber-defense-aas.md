## Overview
This track map is meant to help students plan sequencing and identify the courses we directly support with labs/resources in this repo.

## Recommended sequence (by semester)

### Semester I
- ITNW 1308 — Implementing and Supporting Client Operating Systems
- ITNW 1425 — Fundamentals of Networking Technologies
- ITSC 1316 — Linux Installation and Configuration
- Select 1 course from the Communication Core (catalog link)

### Semester II
- COSC 1436 — Programming Fundamentals I
- ITSY 1342 — Information Technology Security
- ITSY 2443 — Computer System Forensics  ← supported
- ITNW 2412 — Routers

### Semester III
- (see catalog for the full plan; this repo map focuses on supported courses)

### Semester IV
- ITSY 2301 — Firewalls and Network Security  ← supported
- ITSC 2325 — Advanced Linux **or** COSC 1437 — Programming Fundamentals II
- ITNW 2405 — Network Administration
- SPCH 1321 — Business and Professional Communication

### Semester V
- Select 1 course from Language, Philosophy & Culture Core (catalog link)
- ITSY 2430 — Intrusion Detection  ← supported
- ITSY 2442 — Incident Response and Handling
- ITSY 2286 — Internship — Computer and Information Systems Security  ← supported
